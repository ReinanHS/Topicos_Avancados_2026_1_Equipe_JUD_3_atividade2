# Política de segurança

Se você descobrir uma vulnerabilidade de segurança neste projeto, pedimos que a reporte de forma responsável. **Por favor, não abra issues públicas para vulnerabilidades de segurança.**

### Como reportar

1. Envie um e-mail para *reinangabriel1520@gmail.com* com o assunto `[SEGURANÇA] Descrição breve`.
2. Inclua as seguintes informações:
   - Descrição detalhada da vulnerabilidade
   - Passos para reproduzir o problema
   - Impacto potencial
   - Sugestão de correção, se houver
